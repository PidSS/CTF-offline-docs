使用Minisforum UM560小主机办公一周体验

phithon
2022 七月 24 16:21
阅读：34248
心得与体会
好物推荐, homelab, 小主机

本文推荐的UM560非与厂商合作推广，仅为我自己的使用体验。

我大四买了人生里第一台Macbook pro，从那会开始Mac卡顿的问题就一直困扰我到现在。当然这也可能是我自己的问题——我没钱买顶配Macbook，以前公司发的也是低配Macbook pro，我甚至用8G内存的13寸Mac直到2018年才换。

来新公司以后总算摸到了15寸Mac，但还是16G内存Intel芯片的老电脑，被Burpsuite、Jetbrains、Chrome几个软件分分钟教做人，Docker一开风扇就狂转，虚拟机更是想都别想。

好在上半年我们有大部分时间在家办公，家里的台式机暂时性得掩盖了这个问题。但公司这几个月逐渐恢复WFO，一周需要去办公室至少3天，于是我又不得不开始思考如何改善办公效率的问题。

我在国内的时候就关注了minisforum这家小主机品牌商，当时是想组一个homelab，不过后来捡垃圾600块配了一台联想ThinkCentre M73四代U的小主机，所以对他们家一直只是观望。

今年6月的时候看到他们家出了一款新主机UM560，用的是今年AMD发布的新CPU 5625U，预售价格在359美元（国内价格不到2000人民币）。这个6核12线程CPU性能不算特别强，仍然是Zen 3架构，内存也只支持DDR4，但我看中的主要是它的低功耗——TDP只有15W。也就是说，即使我不习惯用这台机器办公，也可以作为homelab替换我现在的渣联想。

犹豫了好几天，最后还是狠下心在官网花359美元买了。官网价格其实没有京东好，但京东买到新加坡运费和税费也不少，所以我还是选择不折腾。如果读者在国内生活，那自然是最方便的。

到手体验

当时买的时候说是预售，最迟会在7月底发货，我大概是7月中旬就收到了。

我买的是裸机，自己另外配了一个32G的内存和512G的NVME硬盘，内存是在Amazon花150新币新买的，硬盘是几年前的吃灰货。最近赶上内存降价，其实也算是个买新机器的好时机。

到手的第一感觉真的是小：

我是有好几年掏垃圾的经验了，除了ThinkCentre M73我之前还有一台J1900的工控机器做软路由、一台J1900 游娱宝盒，而这台UM560是我这么多机器里最小最轻的了，有种垃圾佬突然暴富的感觉。

机器是全塑料材质，相比于金属外壳的M73轻很多，内部有风扇散热，5625U功耗也不高，平时几乎静音（此处鞭尸Intel Macbook Pro一次）。

这台机器还有一点让我非常喜欢的是直接用type-c接口供电，厂商送了一个65W的氮化镓充电头，但我感觉可能换一个更小的20W的充电头，这样电源也能压缩到非常mini。

打开外壳的体验不是特别好，因为需要把底下的塑胶脚贴撕下来才能看到螺丝，很难撕，等后面装回去又没原装那么紧了。

内部做工还不错，很紧致，我只插了一块32G的内存，之后如果做homelab还可以再买一块做双通道：

安装系统

装好内存和硬盘以后，我还没想好办公的系统是用Windows 10、11还是Ubuntu，正好U盘上有一个Windows 10的镜像，于是就装了上去。

装好系统以后显卡分辨率都没啥问题，但没有有线网络、wifi和蓝牙，看来应该是型号太新，Windows镜像里没有相关驱动。于是下了一个鲁大师，先识别下网卡是哪家的。

具体配置如下：

电脑型号               BESSTAR TECH LIMITED UM560 台式电脑
操作系统               Windows 10 专业版 64位（Version 21H1 / DirectX 12）

处理器                 AMD Ryzen 5 5625U with Radeon Graphics 六核
主板                   BESSTAR TECH F6BFC（AMD PCI 标准主机 CPU 桥）
显卡                   AMD Radeon  Graphics ( AMD )
内存                   32 GB ( 英睿达 DDR4 3200MHz 32GB )
主硬盘                 金士顿 SA2000M8500G (500 GB / 固态硬盘)
显示器                 飞利浦 PHL0946 PHL 275B1 ( 27 英寸  )
声卡                   AMD High Definition Audio 控制器
网卡                   英特尔 Ethernet Controller  I225-V

然后去Intel官网下了I225-V的驱动装上就可以连有线了。

连上有线，我想再用鲁大师帮我直接安装一下无线和蓝牙的驱动，但垃圾鲁大师没识别出来，根本没找到驱动，最后还是我打开后盖看了无线网卡的厂商和型号以后自己去官网下的。

无线网卡是MediaTek MT7921，驱动如下：

  • Wifi：https://support.lenovo.com/ec/en/downloads/ds552327-mediatek-mt7921-wireless-lan-driver-for-windows-10-version-2004-or-later-thinkpad-l14-gen-2-types-20x5-20x6-l15-gen-2-types-20x7-20x8
  • 蓝牙：https://support.lenovo.com/lu/en/downloads/ds552065-mediatek-mt7921-bluetooth-driver-for-windows-10-version-2004-or-later-thinkpad-l14-gen-2-l15-gen-2

装好驱动后续使用Windows10就没遇到太多问题。

试用完Windows 10以后我又尝试了一下Windows 11，没想到驱动情况更糟糕，这次连显卡驱动都没有了，打开就是800x600的低分辨率桌面。同样的方法又去网上找了AMD的显卡驱动。

这两顿折腾，给我的感觉就是，因为机器很新，导致一些驱动没法自动安装，需要手工折腾一些时间，但驱动装好以后就与正常机器无异了。

但随之而来的就是，我开始担心如果安装Linux系统，会不会驱动更难找更难折腾？

很快我就被Ubuntu啪啪打脸——装好Ubuntu Desktop 22.04以后，没有遇到任何驱动问题，显卡、有线、无线、蓝牙开箱即用😂。

办公体验

当然，驱动没有问题不代表系统使用没有问题。

因为我是第一次用Ubuntu Desktop办公，所以需要适应很多东西，包括新的输入法、新的Terminal等等。

我装的中文输入法是Rime，它是基于ibus的一个开源输入法框架，通过YAML配置文件进行配置，自定义程度非常高，十分方便。

我装的终端是Terminator，替代了我Mac下的iterm2+tmux方案，其实我还是更喜欢后者，但iTerm2支持在tmux中直接用鼠标复制文字，而Ubuntu下我还没找到类似的方案，所以最终我放弃了tmux。Terminator的优点就是能够分屏，快捷键也可以方便地设置。

另外两个我重度需要的功能是剪切板历史和截图，截图我就用了Ubuntu自带的截图功能，缺点就是截图完后不支持直接在上面添加文字、加马赛克；剪切板历史我使用了CopyQ这款软件，勉强能实现剪切板历史的功能吧，但和Windows下的ditto和Mac下的Paste自然没法比。

剩下一些我重度使用的软件，比如Typora、群晖、Jetbrains、vscode等都有Linux版，直接迁移即可。

到写文章这会，我大概用这一套方案了一个礼拜，中间没有遇到让我弃坑的大问题。相反，整个电脑的流畅度，真心是比Mac要高不少，平时不再需要经常重启电脑，也告别了卡顿现象，32G内存也够我开所有我想用的软件，十分满足~

所以，如果你有和我类似的需求，可以考虑在他们家京东旗舰店买起来。不过，如果你并没有这样的需求，只是因为看到我写的文章一时兴起，建议还是先冷静，思考思考自己是否真的需要一台小主机，毕竟今年经济不是很好，该省钱啦~
