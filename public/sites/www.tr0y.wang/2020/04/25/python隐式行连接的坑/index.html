一些思考

Python 有很多很好使的特性，但是有一些实在是让人心情复杂。比如多行注释，别人家的语言一般都有特定的多行注释，比如 /**/、<!----> 等，而 Python 使用三个连续的单引号 ''' 或者三个连续的双引号 """ 注释多行内容，实际上就是定义了一个字符串，只不过是没用上而已，照这个思路，单行注释还可以这样呢："这是一个注释"。你说有什么问题吧，似乎一下也说不上来；说没问题吧，就是感觉怪怪的，至少本文提到的问题就是多行注释引起的。
